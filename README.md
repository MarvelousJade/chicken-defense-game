# Chicken Defense Game

A fast-paced arcade shooter where you defend your base from waves of invading chickens. Built with **C++20** and **SDL2**.

## Gameplay Preview

https://github.com/user-attachments/assets/a9733ac8-11c0-48f4-980d-18369e12f393

## Features

- Three enemy types (Slow, Medium, Fast) with unique animations
- Dynamic difficulty scaling over time
- Explosion effects with camera shake
- Score and health system (10 hearts)
- Background music and sound effects

## Tech Stack

- **Language:** C++20
- **Rendering:** SDL2 (hardware-accelerated)
- **Resolution:** 1280x720 at 144 FPS
- **Architecture:** OOP with polymorphic entity system, frame-based animation, and a custom timer/camera system

## Controls

| Input | Action |
|---|---|
| Mouse | Aim cannon |
| Left Click | Shoot |
| ESC | Quit |
